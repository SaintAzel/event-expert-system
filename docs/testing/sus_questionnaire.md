# System Usability Scale (SUS) - Evalio

## Instruksi Responden

Gunakan aplikasi Evalio untuk melakukan simulasi evaluasi kesiapan event. Setelah selesai, isi 10 pernyataan berikut dengan skala 1 sampai 5.

Skala:

- 1 = Sangat tidak setuju
- 2 = Tidak setuju
- 3 = Netral
- 4 = Setuju
- 5 = Sangat setuju

## Kuesioner SUS

| No | Pernyataan | Skor 1-5 |
|---|---|---|
| 1 | Saya merasa akan sering menggunakan aplikasi ini untuk mengevaluasi kesiapan event. | |
| 2 | Saya merasa aplikasi ini terlalu rumit untuk digunakan. | |
| 3 | Saya merasa aplikasi ini mudah digunakan. | |
| 4 | Saya membutuhkan bantuan teknis untuk menggunakan aplikasi ini. | |
| 5 | Saya merasa fitur-fitur aplikasi ini terintegrasi dengan baik. | |
| 6 | Saya merasa terdapat terlalu banyak inkonsistensi dalam aplikasi ini. | |
| 7 | Saya membayangkan sebagian besar pengguna dapat mempelajari aplikasi ini dengan cepat. | |
| 8 | Saya merasa aplikasi ini canggung atau sulit digunakan. | |
| 9 | Saya merasa percaya diri saat menggunakan aplikasi ini. | |
| 10 | Saya perlu belajar banyak hal sebelum dapat menggunakan aplikasi ini. | |

## Rumus Perhitungan

- Untuk pernyataan ganjil: skor kontribusi = skor responden - 1.
- Untuk pernyataan genap: skor kontribusi = 5 - skor responden.
- Total seluruh kontribusi dikalikan 2,5.
- Skor SUS akhir berada pada rentang 0 sampai 100.
